Relação com a Linguagem SQL



Como o SQL é usado

A gente pode usar comandos SQL comuns como SELECT, INSERT, UPDATE, DELETE, além dos comandos de criação de tabelas e controle de permissões. O diferencial é que no Oracle dá pra usar PL/SQL pra criar blocos de código mais complexos, como funções e procedimentos.

Comandos compatíveis

O Oracle suporta todos os tipos de comandos:

DDL: criar e alterar tabelas (CREATE, DROP, ALTER) DML: manipular dados (SELECT, INSERT, UPDATE, DELETE) DCL: controlar permissões (GRANT, REVOKE) TCL: controlar transações (COMMIT, ROLLBACK)
