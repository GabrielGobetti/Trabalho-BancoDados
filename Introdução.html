Como foi Criado

  • O Oracle Database foi criado no final dos anos 1970
  • Outros modelos: Embora seja predominantemente relacional, o Oracle Database evoluiu para suportar outros modelos de dados, incluindo:
  • Criado por Larry Ellison, Bob Miner e Ed Oates
  • A ideia surgiu quando Ellison encontrou um protótipo funcional de um banco de dados relacional e percebeu que nenhuma empresa havia comercializado essa tecnologia ainda.
  • A primeira versão comercial do Oracle Database foi lançada em 1979
  • desde então, ele evoluiu significativamente, tornando-se um dos sistemas de gerenciamento de banco de dados mais utilizados no mundo.


Versão mais recente

  • A versão mais recente do Oracle Database é a Oracle Database 23ai
  • lançada em maio de 2024.
  • Essa versão traz novos recursos voltados para inteligência artificial e está disponível tanto na nuvem quanto para instalações locais.
